# Word to PDF Converter 📝➡️📄

Convert your Word documents to PDF quickly and easily with this sleek web app! Built with modern web technologies, this app gives you a seamless experience from file selection to download.

## 🔥 Features
* Upload your Word documents and convert them to PDF instantly.
* Automatically downloads the converted PDF.
* Clean and responsive UI design.
* Fully deployed with working frontend and backend.

## 💻 Technologies Used
* **Node.js** – Runs JavaScript on the server-side.
* **Express.js** – Node.js framework for backend API handling.
* **Multer** – Middleware for handling file uploads.
* **React.js** – JavaScript library for building the frontend.
* **Tailwind CSS** – Modern UI design and styling.

## 🚀 How to Use
1. Click on **Select Document**.
2. Choose the Word file from your device.
3. The file will show as selected.
4. Click on **Convert File**.
5. Your PDF will be automatically downloaded.
